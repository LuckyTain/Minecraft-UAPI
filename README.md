# Minecraft - UsefulAPI
==============
一个为了方便开发Minecraft服务端(for BukkitAPI)插件的API。

用途/功能
--------
```
* 1 用户
  * 1.1 额外的API                    [100%]
  * 1.2 技能管理器                   [100%]
  * 1.3 GUI数据管理器                [50%]
  * 1.4 自定义数据管理器             [0%]
* 2 方块
  * 2.1 方块检查器                   [100%]
  * 2.2 多结构方块检查器             [100%]
* 3 道具
  * 3.1 道具检查器                   [100%]
  * 3.2 道具创建工具                 [100%]
  * 3.3 NBT修改工具                  [90%]
* 4 网络
  * 4.1 自定义封包发送               [100%]
  * 4.2 封包拦截事件                 [0%]
* 5 NBT
  * 5.1 封装的NBT类                  [100%]
* 6 GUI
  * 6.1 自定义GUI                    [50%]
* 7 插件
  * 7.1 自定义规则                   [100%]
  * 7.2 自定义事件                   [100%]
* 8 其他工具
  * 8.1 粒子效果                     [100%]
  * 8.2 底层类工具                   [100%]
  * 8.3 颜色工具                     [10%]
```
